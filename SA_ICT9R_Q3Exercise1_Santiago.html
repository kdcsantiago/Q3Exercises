HTML

HyperText Markup Language

Hypertext Markup Language, is a standard file format for creating web pages. It defines the structure and layout of a web page and is composed of elements and tags that can be used to create headings, paragraphs, etc. HTML is used in conjunction with other technologies such as CSS to create dynamic, interactive web pages.

CSS

Cascading Style Sheets

CSS is a language for describing the presentation of a markup-language-written document. It is used to put designs and styles on web pages such as the colors, the fonts, the layouts of the letters, and the spacing. The types of CSS are inline CSS, internal CSS, external CSS.
